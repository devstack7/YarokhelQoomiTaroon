## تصویر کے لیے Options:

### Option 1: Placeholder استعمال کریں
```
https://via.placeholder.com/400x500/2c5f2d/ffffff?text=صاحب+خان+عاجز
```

### Option 2: مقامی تصویر اپ لوڈ کریں

### Option 3: کوئی موجودہ URL استعمال کریں
- اگر آپ کے پاس پہلے سے کوئی image URL ہے

## 🎯 کیسے Add کریں:

1. **Admin Panel کھولیں:** `http://localhost:3000/admin/dashboard`
2. **Login کریں** (اگر نہیں کیا)
3. **"نیا اضافہ کریں"** بٹن پر کلک کریں
4. **اوپر کی معلومات copy paste کریں**
5. **تصویر:**
   - یا تو URL paste کریں
   - یا File upload کریں
6. **"محفوظ کریں"** دبائیں
