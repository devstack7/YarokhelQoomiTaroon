## 🖼️ تصاویر بعد میں کیسے Add کریں؟

### Option 1: تصویر فائل Upload کریں

1. Admin Dashboard کھولیں
2. جس leader کی تصویر add کرنی ہے، اس کے سامنے ✏️ (Edit) بٹن دبائیں
3. "تصویر اپ لوڈ کریں" کے نیچے **Choose File** بٹن سے تصویر select کریں
4. **"محفوظ کریں"** دبائیں

### Option 2: تصویر URL استعمال کریں

اگر تصویر پہلے سے کسی website پر ہے:

1. Admin Dashboard کھولیں
2. ✏️ Edit بٹن دبائیں
3. "تصویر URL" field میں link paste کریں
4. **"محفوظ کریں"** دبائیں

---

## 🌐 تصاویر Online Upload کرنے کے لیے

اگر آپ کے پاس تصاویر ہیں اور آپ انہیں online upload کرنا چاہتے ہیں:

### ImgBB (آسان اور مفت)

1. **https://imgbb.com** پر جائیں
2. **"Start uploading"** کلک کریں
3. تصویر select کریں
4. Upload ہونے کے بعد **"Direct link"** copy کریں
5. Admin Panel میں paste کریں

### Imgur (بھی مفت)

1. **https://imgur.com** پر جائیں
2. **"New post"** کلک کریں
3. تصویر upload کریں
4. تصویر پر right-click → **"Copy image address"**
5. Admin Panel میں paste کریں

## 🔄 Leader کو Edit یا Delete کرنا

### Edit کرنے کے لیے:
1. Admin Dashboard کھولیں
2. جس leader کو edit کرنا ہے، اس کے سامنے ✏️ بٹن دبائیں
3. تبدیلیاں کریں
4. **"محفوظ کریں"** دبائیں

### Delete کرنے کے لیے:
1. Admin Dashboard کھولیں
2. جس leader کو delete کرنا ہے، اس کے سامنے 🗑️ بٹن دبائیں
3. Confirm کریں

## 💡 Pro Tips

1. **پہلے بغیر تصویر کے سب add کریں** - تیز ہوگا
2. **پھر ایک ایک کر کے تصاویر update کریں** - آرام سے
3. **تفصیل چھوٹی رکھیں** - 2-3 جملے کافی ہیں
4. **Placeholder images استعمال کریں** - جب تک اصل تصاویر نہ ملیں

## ❓ مدد چاہیے؟

اگر کوئی مسئلہ ہو:

1. **F12 دبا کر Console دیکھیں** - کوئی error ہے؟
2. **Refresh کریں** (Ctrl+R) - کبھی کبھی کام کر جاتا ہے
3. **Logout اور پھر Login کریں** - session refresh ہو جاتا ہے
