# یارُوخیل قومی تڑون - Yaru Khel Qoomi Taroon

ضلع تورغر، مانسہرہ کی قومی تنظیم کی آفیشل ویب سائٹ

## تنصیب (Installation)

```bash
# Dependencies install کریں
npm install

# Development server شروع کریں
npm start

# Production build بنائیں
npm run build
```

## Vercel پر Deploy کرنے کا طریقہ

1. [Vercel](https://vercel.com) پر اکاؤنٹ بنائیں
2. GitHub repository بنا کر اس code کو push کریں
3. Vercel dashboard میں "Import Project" کریں
4. Repository منتخب کریں
5. Deploy کریں!
