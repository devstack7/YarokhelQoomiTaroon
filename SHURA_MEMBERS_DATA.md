# 🕌 مجلس شوریٰ اراکین کی تفصیلات

## Database Columns:
- **name** - رکن کا نام
- **address** - پتہ (سب کے لیے "پاکستان")
- **detail** - مکمل تفصیل
- **display_order** - ترتیب نمبر (1-16)
- **image_url** - تصویر کا URL

## 🎯 استعمال:

### Admin Panel میں Add کرنے کے لیے:
1. `http://localhost:3000/admin/shura` کھولیں
2. "نیا رکن شامل کریں" دبائیں
3. اوپر سے copy paste کریں
4. Save کریں

### CSV Import کے لیے:
- `shura_members.csv` فائل استعمال کریں
- Database columns: `name`, `address`, `detail`, `display_order`, `image_url`
