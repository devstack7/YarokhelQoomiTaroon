# 📂 فنڈ کی اقسام - استعمال کی مکمل رہنمائی

## مکمل ہو گیا ✅

**فنڈ کی متحرک اقسام** کا نظام مکمل طور پر تیار ہے۔ اب آپ جتنی چاہیں categories بنا سکتے ہیں۔

---

## 🎯 خصوصیات

### ✅ مکمل شدہ خصوصیات:
1. **نئی Category بنانا** - جتنی چاہیں بنائیں
2. **Category میں ترمیم** - نام، رنگ، آئیکن تبدیل کریں
3. **Category کو فعال/غیر فعال کرنا** - بغیر حذف کیے چھپا دیں
4. **Category کو حذف کرنا** - جس میں transactions نہیں
5. **Dashboard میں Category-wise خلاصہ** - ہر category کا الگ حساب
6. **Transaction میں Category منتخب کرنا** - ڈراپ ڈاؤن سے
7. **Category کی بنیاد پر فلٹر کرنا** - صرف ایک category دیکھیں
8. **Viewer میں Category دیکھنا** - مکمل سپورٹ

---

## 🚀 استعمال کی رہنمائی

### 1️⃣ نئی Category کیسے بنائیں

#### قدم بہ قدم:
1. **Manager login** کریں: `funds.manager@yarukhelqoomi.com`
2. **Dashboard** پر جائیں
3. **"📂 فنڈ کی اقسام"** بٹن دبائیں
4. **"➕ نئی قسم شامل کریں"** دبائیں
5. **فارم بھریں:**
   - **نام (اردو)**: مثال: `تعلیمی فنڈ`
   - **آئیکن**: 📚 یا کوئی بھی emoji
   - **رنگ**: کسی بھی رنگ کو منتخب کریں
   - **ترتیب نمبر**: چھوٹا نمبر پہلے دکھائی دے گا
   - **تفصیل**: اختیاری
6. **"محفوظ کریں"** دبائیں

#### مثال Categories:
- 📚 **شجرنسب اور گروپ** - #4caf50 (سبز)
- 🎉 **ایونٹ** - #9c27b0 (جامنی)
- 🏥 **طبی امداد** - #f44336 (سرخ)
- 📖 **مذہبی کام** - #2196f3 (نیلا)
- ⚽ **کھیل کود** - #ff9800 (نارنجی)
- 🏠 **عمارت** - #795548 (بھورا)

---

### 2️⃣ Transaction میں Category کا استعمال

#### آمدن (Income) کے لیے:
1. Dashboard پر **"➕ نیا"** دبائیں
2. **قسم**: آمدن منتخب کریں
3. **فنڈ کی قسم**: مطلوبہ category منتخب کریں (جیسے: شجرنسب اور گروپ)
4. **شخص منتخب کریں**: عطیہ دہندہ کا نام
5. باقی تفصیلات بھریں
6. **"محفوظ کریں"** دبائیں

#### خرچہ (Expense) کے لیے:
1. Dashboard پر **"➕ نیا"** دبائیں
2. **قسم**: خرچہ منتخب کریں
3. **فنڈ کی قسم**: منتخب کریں (جس میں سے رقم کٹے گی)
4. ⚠️ **اہم**: جو category منتخب کریں گے، اسی میں سے رقم کم ہوگی
5. **خرچہ کی تفصیل** لکھیں (جیسے: کرایہ، بجلی، پانی)
6. باقی تفصیلات بھریں
7. **"محفوظ کریں"** دبائیں

---

### 3️⃣ Dashboard پر Category-wise خلاصہ دیکھنا

Dashboard کھولتے ہی آپ کو **تین قسم کے Cards** نظر آئیں گے:

---

### 4️⃣ Category کی بنیاد پر فلٹر کرنا

Dashboard یا Viewer میں:
1. **"فنڈ:"** والے section میں جائیں
2. کسی **category** پر کلک کریں (جیسے: 📚 شجرنسب اور گروپ)
3. صرف اس category کی transactions دکھائی دیں گی
4. **"تمام"** دبانے سے سب کچھ واپس دکھے گا

---

### 5️⃣ Category میں ترمیم کیسے کریں

1. **"📂 فنڈ کی اقسام"** پر جائیں
2. جس category میں تبدیلی کرنی ہے، اس کے **✏️ بٹن** پر کلک کریں
3. تبدیلیاں کریں:
   - نام بدلیں
   - آئیکن بدلیں
   - رنگ بدلیں
   - ترتیب نمبر بدلیں
4. **"محفوظ کریں"** دبائیں

---

### 6️⃣ Category کو چھپانا (غیر فعال کرنا)

اگر کوئی category **استعمال نہیں کرنی** لیکن حذف بھی نہیں کرنا:

1. **"📂 فنڈ کی اقسام"** پر جائیں
2. اس category کے **⊗ بٹن** پر کلک کریں
3. Category **غیر فعال** ہو جائے گی
4. یہ ڈراپ ڈاؤن میں نظر نہیں آئے گی
5. دوبارہ **✓ بٹن** دبانے سے فعال ہو جائے گی

---

### 7️⃣ Category کو حذف کرنا

⚠️ **احتیاط**: صرف وہ categories حذف ہو سکتی ہیں جن میں کوئی transaction نہیں۔

1. **"📂 فنڈ کی اقسام"** پر جائیں
2. اس category کے **🗑️ بٹن** پر کلک کریں
3. تصدیق کریں
4. اگر اس میں transactions ہیں تو حذف نہیں ہوگی

**تجویز**: حذف کرنے کی بجائے **غیر فعال** کر دیں۔

---

### 8️⃣ Viewer میں Categories کیسے دکھتی ہیں

**Viewer login**: `funds.viewer@yarukhelqoomi.com`

Viewer کو بھی **مکمل category support** ہے:
- ✅ Category-wise summary cards
- ✅ Category filters (تمام فنڈز دیکھیں)
- ✅ Transaction میں category badge
- ❌ Categories میں ترمیم نہیں کر سکتے (صرف Manager)

## ❓ عام سوالات (FAQ)

### Q1: کیا ایک transaction میں 2 categories ہو سکتی ہیں؟
**جواب**: نہیں، ہر transaction صرف **ایک category** میں ہو سکتی ہے۔

### Q2: کیا category حذف ہونے سے پرانی transactions بھی حذف ہوں گی؟
**جواب**: نہیں۔ اگر category میں transactions ہیں تو وہ **حذف نہیں ہوگی**۔ پہلے غیر فعال کریں۔

### Q3: کیا Viewer categories بنا سکتا ہے؟
**جواب**: نہیں، صرف **Manager** ہی categories بنا، ترمیم، یا حذف کر سکتا ہے۔

### Q4: کتنی categories بنا سکتے ہیں؟
**جواب**: **لامحدود**۔ جتنی چاہیں بنائیں۔

### Q5: Category کا رنگ کیا کرتا ہے؟
**جواب**: Dashboard اور Viewer میں summary cards کا رنگ تبدیل ہوتا ہے۔

### Q6: کیا default categories (شجرنسب، ایونٹ) ضروری ہیں؟
**جواب**: نہیں، آپ انہیں بھی ترمیم یا غیر فعال کر سکتے ہیں۔

## 🔧 تکنیکی تفصیلات

### Database Table: `fund_categories`
```sql
- id (uuid)
- name (text) - انگریزی نام (auto-generated)
- name_urdu (text) - اردو نام
- icon (text) - emoji
- color (text) - hex color code
- description (text) - تفصیل
- display_order (integer) - ترتیب
- is_active (boolean) - فعال/غیر فعال
- created_at (timestamp)
```

### Integration:
- ✅ FundsDashboard.js
- ✅ FundsViewer.js
- ✅ CategoryManagement.js
- ✅ fund_transactions table (category column)
